
*] Basic   commands  :

1-  flutter analyze  => to discover and display issue if the app segmeted inot levels  

2-  flutter test =>  testing the app using the  {widget_test.dart} file 

3-  flutter build apk --release --split-per-abi  =>  for buidling several apk publish Edition  suits : [ online platforms  : gumRoadk , Code Canyon ] 

4-  flutter build apk --release  =>  for buidling only one apk publish Edition  suits : [ Solo usage ] 

5-  flutter build appbundle --release   => for building publish into for [Google Play , App store ]
-------------------------

*]  Featured commands (بناء مع كل التحسينات) :

flutter build apk --release \
  --split-per-abi \
  --obfuscate \
  --split-debug-info=./debug_info/ \
  --dart-define=APP_ENV=production \
  --no-tree-shake-icons 

Features :  
--obfuscate: يحمي كودك من الهندسة العكسية

--split-debug-info: يحفظ معلومات التصحيح منفصلة

--dart-define: متغيرات بيئة التطبيق

--no-tree-shake-icons: يحفظ كل الأيقونات

6- creating  project zipped folder:  

# 1. تأكد أنك في مجلد المشروع
cd "E:\My Mob Projects\3 Flutter abo hamza\demos\finance_app"

# 2. إنشاء مجلد الحزمة
